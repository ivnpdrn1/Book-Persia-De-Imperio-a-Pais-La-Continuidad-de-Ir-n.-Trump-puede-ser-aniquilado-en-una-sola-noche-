# 🔧 Cómo convertir esto en TOC automático en Word (KDP)

1. Selecciona cada **“Parte” → Heading 1**
2. Selecciona cada **“Capítulo” → Heading 2**
3. Ve a:

   * Referencias → Tabla de contenido
   * Elige “Automática”

👉 KDP reconocerá esto perfectamente.
