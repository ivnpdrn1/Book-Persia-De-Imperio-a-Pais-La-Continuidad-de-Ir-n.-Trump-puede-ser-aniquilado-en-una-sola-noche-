# Capítulo 24

## El Sistema Actual

**1979–Presente**

**Periodo:** 1979–Presente
Etapa en la que el sistema iraní opera como una estructura híbrida que integra religión, Estado y geopolítica.

**Actor principal:** República Islámica de Irán
Un sistema que combina instituciones políticas con autoridad religiosa.

**Oponente:** Sistema internacional y presiones externas (especialmente Occidente)
Entorno global que condiciona la operación del sistema.

**Estado del sistema:** Sistema consolidado híbrido
El sistema ha logrado sostenerse en el tiempo bajo su propia lógica.

**Mecanismo de estabilidad (paz del sistema):** Integración entre identidad, estructura política y estrategia geopolítica
La estabilidad se logra alineando estos tres elementos.

**Forma de estabilidad:** Híbrida (cohesión interna + adaptación externa)
El sistema combina control interno con flexibilidad estratégica.

**Resultado en la identidad de Irán:** Consolidación de un modelo propio de Estado
El sistema no replica modelos externos, opera bajo su propia lógica.

**Religión (rol en el sistema):** Islam chiita como base estructural del poder
Más del 95 por ciento de la población es musulmana, con fuerte influencia del chiismo en el sistema político.

**Idioma (rol en el sistema):** Persa como cohesión cultural y nacional
Entre el 85 y el 95 por ciento utiliza el persa como idioma principal.

**Recurso principal (base de riqueza):** Petróleo, gas y posición geopolítica
El sistema mantiene relevancia estratégica global.

**Capacidad económica (equivalente PIB):** Media — economía resiliente bajo sanciones
El sistema opera con limitaciones externas, pero mantiene funcionamiento interno.

**Capacidad militar (proyección de poder):** Media-alta — capacidad defensiva y de influencia regional
El sistema combina fuerzas tradicionales con estrategias indirectas.

**Dimensión territorial:** ~1,600,000 km² (~618,000 mi²)
Territorio estable bajo control estatal.

**Población estimada:** ~85 millones
Con alta urbanización (70–75 por ciento urbano).

## Análisis del sistema

En este periodo, el sistema no solo existe.

Funciona.

Después de la revolución, el sistema iraní logra consolidarse como una estructura híbrida. No es una teocracia tradicional, ni un Estado moderno convencional. Es una combinación de ambos.

Esto le da una característica única.

Flexibilidad dentro de su propia lógica.

La estabilidad interna se basa en la integración de identidad y poder. A diferencia del periodo Pahlavi, donde existía una desconexión, el sistema actual mantiene alineación entre estructura política y base social.

Esto le da legitimidad interna.

Pero introduce tensión externa.

El sistema internacional no siempre reconoce esta lógica. Esto genera presión en forma de sanciones, aislamiento y conflicto indirecto. Sin embargo, el sistema ha desarrollado mecanismos para operar bajo estas condiciones.

Aquí aparece un elemento clave.

La resiliencia operativa.

La economía no es dominante, pero es funcional. A pesar de las limitaciones, el sistema mantiene su capacidad de operar. Esto se logra mediante adaptación constante y uso estratégico de recursos.

La capacidad militar no se basa solo en confrontación directa. El sistema combina defensa tradicional con influencia regional. Esto le permite proyectar poder sin exponerse completamente.

La religión sigue siendo el eje central, pero no es el único elemento. Se integra con la política y la estrategia, formando una estructura compleja.

El sistema no es rígido.

Es adaptable dentro de sus propios límites.

Caso conceptual.

Un sistema puede intentar alinearse completamente con el entorno global, perdiendo parte de su identidad, o puede operar bajo su propia lógica, adaptándose solo lo necesario para sobrevivir.

El primer modelo busca integración total.

El segundo busca autonomía funcional.

Eso es lo que ocurre aquí.

La estabilidad deja de depender de aceptación externa.

Depende de coherencia interna.

La lección es clara.

Un sistema que logra operar bajo su propia lógica puede sostenerse incluso bajo presión externa constante.

Para entender a Irán hoy, este momento es fundamental. Representa el resultado de todo el proceso histórico. Un sistema que ha aprendido a adaptarse, a reconstruirse, a influir y a resistir.

Intentar analizarlo como un sistema convencional genera errores.

Pero entender su lógica permite interactuar con él de forma más efectiva.

El sistema no busca desaparecer en el entorno global.

Busca operar dentro de él bajo sus propios términos.

Y eso define el siguiente paso.

La negociación.

No como adaptación.

Sino como interacción entre sistemas distintos.

Ahí se define el futuro.

Y con ello, el último capítulo.
