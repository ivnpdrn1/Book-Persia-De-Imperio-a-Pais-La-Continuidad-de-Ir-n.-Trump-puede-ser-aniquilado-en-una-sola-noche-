# Capítulo 24

## El Sistema Actual

**1979–Presente**

**Periodo:** 1979–Presente. Corresponde a la etapa en la que el sistema iranio opera bajo un modelo híbrido que combina religión, Estado y estrategia geopolítica.

**Actor principal:** República Islámica de Irán. Un sistema que integra estructura política, autoridad religiosa e influencia estratégica.

**Oponente:** Presión internacional y sistemas externos. Especialmente potencias occidentales y dinámicas regionales que buscan limitar su influencia.

**Estado del sistema:** Sistema híbrido consolidado (estructura + identidad + estrategia). El sistema combina múltiples dimensiones para sostenerse.

**Mecanismo de estabilidad (paz del sistema):** Equilibrio entre legitimidad interna e influencia externa. La estabilidad se logra gestionando tensiones en ambos niveles.

**Forma de estabilidad:** Compleja (identidad + estructura + adaptación). El sistema no depende de un solo elemento.

**Resultado en la identidad de Irán:** Consolidación de un modelo propio. El sistema no replica estructuras externas, sino que opera bajo su propia lógica.

**Religión (rol en el sistema):** Islam chiita como eje estructural del Estado. Base de legitimidad y cohesión interna.

**Idioma (rol en el sistema):** Persa como elemento de continuidad cultural y administrativa.

**Recurso principal (base de riqueza):** Petróleo, gas y control estratégico de recursos.

**Capacidad económica (equivalente PIB):** Media. Economía condicionada por sanciones y dinámica internacional.

**Capacidad militar (proyección de poder):** Media-alta. Capacidad de defensa y proyección indirecta.

**Dimensión territorial:** ~1,600,000 km². Estado consolidado.

**Población estimada:** ~80–90 millones.

## Análisis del sistema

En este periodo, el sistema no vuelve a modelos anteriores.

Se convierte en algo distinto.

Después de la Revolución de 1979, el sistema iranio no se limita a reemplazar una estructura por otra. Construye un modelo híbrido que integra elementos que, en otros contextos, operan por separado.

Religión.

Estado.

Estrategia.

Todo dentro de una misma estructura.

Esto genera una forma de poder diferente. El sistema no depende exclusivamente de instituciones políticas tradicionales, ni de una autoridad religiosa aislada, ni de una lógica puramente geopolítica.

Depende de la interacción entre todas.

La legitimidad interna proviene en gran medida de la identidad religiosa. El sistema se presenta como coherente con la base cultural de la población. Esto le permite sostener cohesión incluso en contextos de presión.

Pero esa legitimidad no es suficiente por sí sola.

El sistema también necesita operar externamente.

Aquí aparece la dimensión estratégica. Irán no busca necesariamente control directo en todos los espacios. Utiliza mecanismos indirectos: influencia regional, alianzas, proyección asimétrica.

Esto reduce su exposición.

Y aumenta su capacidad de maniobra.

La economía introduce una limitación. Las sanciones y la presión internacional afectan su capacidad de crecimiento. Pero el sistema se adapta, reorganiza sus recursos y encuentra formas de sostenerse.

No es una economía libre.

Es una economía condicionada.

La capacidad militar refleja esta lógica híbrida. No se trata solo de fuerza convencional, sino de capacidad de operar en múltiples niveles: defensa, disuasión y proyección indirecta.

Aquí aparece el elemento clave.

El sistema como integración de múltiples dimensiones.

Un sistema puede depender de un solo eje —económico, político o militar— lo que lo hace vulnerable si ese eje falla. O puede integrar múltiples dimensiones, lo que le permite compensar debilidades con otras capacidades.

El primero es simple.

El segundo es complejo.

Eso es lo que ocurre aquí.

El sistema iranio no es el más fuerte en cada dimensión.

Pero es consistente en todas.

Como señala Vali Nasr, el modelo iraní contemporáneo combina identidad religiosa, estructura estatal e influencia regional, creando una forma de poder que no puede entenderse desde un solo enfoque¹.

Esto define la etapa.

La estabilidad no depende de eliminar tensiones.

Depende de gestionarlas.

Un sistema complejo no busca equilibrio perfecto.

Busca equilibrio dinámico.

Eso es lo que hace aquí.

El sistema no es estático.

Se ajusta constantemente.

Y en ese ajuste, mantiene su funcionamiento.

Porque la estabilidad no es ausencia de conflicto.

Es capacidad de operar dentro de él.
