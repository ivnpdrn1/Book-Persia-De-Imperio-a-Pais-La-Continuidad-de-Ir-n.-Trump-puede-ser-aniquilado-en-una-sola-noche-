# Capítulo 13

## El Cambio de Sistema (Islam)

**651–900 d.C.**

**Periodo:** 651–900 d.C. Corresponde a la etapa en la que el sistema sasánida colapsa y es reemplazado por un nuevo sistema estructural: el islam.

**Actor principal:** Califatos islámicos. Un sistema completamente distinto que redefine la estructura política, religiosa y social del territorio.

**Oponente:** Sistema sasánida en colapso. Un modelo rígido que no logra adaptarse a las nuevas condiciones.

**Estado del sistema:** Cambio estructural total (reemplazo del sistema). El sistema anterior desaparece como estructura dominante.

**Mecanismo de estabilidad (paz del sistema):** Integración bajo una nueva lógica religiosa y política. La estabilidad se construye sobre un nuevo marco de referencia.

**Forma de estabilidad:** Transformacional (redefinición completa). El sistema no evoluciona, es sustituido.

**Resultado en la identidad de Irán:** Adaptación profunda dentro de un sistema externo. La identidad persa no desaparece, se reconfigura.

**Religión (rol en el sistema):** Islam como nueva estructura central. Más del 80–90 por ciento de la población se integra progresivamente al nuevo sistema religioso.

**Idioma (rol en el sistema):** Árabe como lengua dominante en administración, con persistencia del persa en la cultura.

**Recurso principal (base de riqueza):** Integración en redes económicas más amplias del mundo islámico.

**Capacidad económica (equivalente PIB):** Media-alta. El sistema se beneficia de una red económica más extensa.

**Capacidad militar (proyección de poder):** Alta. El nuevo sistema tiene capacidad de expansión rápida y sostenida.

**Dimensión territorial:** Integrado en un sistema mucho más amplio que supera los 10,000,000 km².

**Población estimada:** ~30–40 millones en la región iraní, dentro de un sistema mayor.

## Análisis del sistema

En este periodo, el sistema no se adapta.

Es reemplazado.

Después de un desgaste prolongado, el sistema sasánida pierde su capacidad de sostenerse. No logra ajustarse a las nuevas condiciones, y cuando aparece un sistema diferente, con una lógica distinta, no puede absorberlo.

El resultado no es una transición.

Es una ruptura.

El islam introduce una estructura completamente nueva. No solo redefine el poder político, sino también la base cultural, religiosa y social. El sistema ya no opera bajo la lógica anterior.

Opera bajo otra.

Este es un punto crítico.

El cambio no ocurre porque el sistema anterior sea débil en todos sus aspectos. Ocurre porque no puede adaptarse a una lógica que no entiende.

El nuevo sistema no solo conquista.

Integra.

Pero lo hace desde una base distinta.

La religión se convierte en el eje central. A diferencia del zoroastrismo institucional del sistema sasánida, el islam no solo estructura el poder, sino que redefine la comunidad. Introduce una lógica de pertenencia más amplia.

El idioma también cambia. El árabe se convierte en lengua dominante en administración y conocimiento, pero el persa no desaparece. Permanece en la cultura, en la literatura y en la identidad.

Esto es clave.

Porque el sistema persa no se elimina.

Se transforma dentro del nuevo sistema.

La economía se expande al integrarse en una red más amplia. El territorio deja de ser un sistema independiente y pasa a formar parte de una estructura mayor.

La capacidad militar del nuevo sistema es superior en términos de velocidad y cohesión. Esto refuerza el cambio.

Aquí aparece el elemento clave.

El reemplazo por incompatibilidad estructural.

Un sistema puede adaptarse a cambios graduales. Pero cuando enfrenta una lógica completamente distinta, la adaptación puede no ser posible.

En ese caso, ocurre el reemplazo.

Eso es lo que sucede aquí.

El sistema sasánida no colapsa solo por desgaste.

Colapsa porque no puede integrar el nuevo modelo.

Como señala Ervand Abrahamian, la transición hacia el sistema islámico no implica la desaparición de la identidad persa, sino su reconfiguración dentro de una nueva estructura política y religiosa¹.

Esto define la etapa.

El sistema cambia.

Pero la identidad permanece.

La estabilidad deja de depender de continuidad estructural.

Depende de capacidad de adaptación dentro de un sistema externo.

Un sistema puede desaparecer completamente.

O puede transformarse y persistir dentro de otro.

El sistema persa elige lo segundo.

Y eso cambia todo.

Porque a partir de este momento, el sistema ya no opera como entidad independiente.

Opera desde dentro.

Y desde ahí, comienza una nueva fase.
