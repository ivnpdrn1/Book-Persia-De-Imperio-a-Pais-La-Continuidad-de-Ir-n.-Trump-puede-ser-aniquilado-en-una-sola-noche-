# Capítulo 22

## La Dinastía Pahlavi

**1925–1979 d.C.**

**Periodo:** 1925–1979 d.C. Corresponde a la etapa en la que el sistema iranio intenta recuperar control interno mediante un proceso de modernización acelerada.

**Actor principal:** Dinastía Pahlavi. Un sistema que busca reconstruir el Estado con base en centralización y modernización.

**Oponente:** Estructuras tradicionales internas y presión externa persistente. El sistema enfrenta resistencia tanto dentro como fuera.

**Estado del sistema:** Sistema en reconstrucción centralizada (control con transformación). El sistema recupera capacidad, pero cambia su forma interna.

**Mecanismo de estabilidad (paz del sistema):** Modernización estructural. La estabilidad se busca mediante desarrollo, centralización y control institucional.

**Forma de estabilidad:** Impositiva (estructura + cambio acelerado). El sistema intenta transformar la sociedad desde el poder.

**Resultado en la identidad de Irán:** Tensión entre modernización y tradición. Se genera una fractura entre el sistema político y la base social.

**Religión (rol en el sistema):** Reducción del rol religioso en la estructura del Estado. Se intenta desplazar su influencia política.

**Idioma (rol en el sistema):** Persa como herramienta de identidad nacional modernizada.

**Recurso principal (base de riqueza):** Petróleo y recursos energéticos. Se convierten en la base económica del sistema.

**Capacidad económica (equivalente PIB):** Media-alta. Crecimiento impulsado por recursos energéticos y modernización.

**Capacidad militar (proyección de poder):** Media-alta. Fuerzas modernizadas con apoyo externo.

**Dimensión territorial:** ~1,600,000 km². Estado definido con control central.

**Población estimada:** ~30–40 millones.

## Análisis del sistema

En este periodo, el sistema intenta recuperar el control.

Pero lo hace cambiando su lógica.

Después de una etapa prolongada de influencia externa y dependencia, el sistema iranio busca reconstruir su autonomía. La dinastía Pahlavi introduce un proceso de modernización que tiene un objetivo claro.

Fortalecer el Estado.

Esto implica centralizar el poder, reorganizar la economía, modernizar las instituciones y reducir la influencia de estructuras tradicionales.

El sistema busca alinearse con modelos externos.

Especialmente occidentales.

Esto le permite aumentar su capacidad en términos de infraestructura, economía y fuerza militar. El Estado se vuelve más eficiente en muchos aspectos.

Pero introduce una nueva tensión.

Desconexión.

La modernización no se construye desde la base social. Se impone desde el centro. Esto genera una diferencia entre el sistema político y la sociedad.

El sistema funciona.

Pero no se alinea.

La religión, que había sido un elemento clave de cohesión durante siglos, es desplazada del centro del poder. Esto reduce su influencia institucional, pero no elimina su presencia social.

La identidad no desaparece.

Se separa del sistema.

La economía crece, impulsada por el petróleo. Esto permite financiar el proceso de modernización. Pero también genera dependencia de un recurso y refuerza la conexión con actores externos.

La capacidad militar se fortalece, pero en gran medida depende de apoyo externo. El sistema gana poder, pero no completamente independiente.

Aquí aparece el elemento clave.

Modernizar sin integrar.

Un sistema puede transformarse incorporando cambios de forma progresiva, alineando estructura y sociedad. O puede imponer cambios desde el centro, lo que genera resistencia.

El primero evoluciona.

El segundo tensiona.

Eso es lo que ocurre aquí.

El sistema Pahlavi no es débil.

Pero no es estable.

Como señala Nikki R. Keddie, los procesos de modernización en Irán durante el siglo XX generaron avances significativos en la estructura del Estado, pero también produjeron tensiones profundas entre el poder político y la sociedad¹.

Esto define la etapa.

La estabilidad deja de depender solo de control o desarrollo.

Depende de alineación.

Un sistema puede ser fuerte estructuralmente y aún así ser inestable si no logra conectar con su base social.

Eso es lo que ocurre aquí.

El sistema crece.

Pero la tensión también.

Y esa tensión no desaparece.

Se acumula.

Hasta que se convierte en algo distinto.

No en ajuste.

Sino en ruptura.

Eso es lo que prepara el siguiente momento.
