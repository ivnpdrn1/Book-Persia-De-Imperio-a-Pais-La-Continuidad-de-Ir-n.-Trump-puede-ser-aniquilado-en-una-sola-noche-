# Capítulo 23

## La Revolución Iraní

**1979 d.C.**

**Periodo:** 1979 d.C.
Momento de ruptura total donde el sistema redefine su estructura en respuesta a la desconexión entre poder e identidad.

**Actor principal:** Movimiento revolucionario liderado por Ruhollah Jomeini
Una fuerza que articula identidad, religión y oposición al modelo existente.

**Oponente:** Sistema Pahlavi (estructura modernizada sin cohesión interna)
Un modelo percibido como desconectado de la identidad del sistema.

**Estado del sistema:** Sistema en ruptura (colapso del modelo anterior)
La estructura existente pierde legitimidad y capacidad de sostenerse.

**Mecanismo de estabilidad (paz del sistema):** Reintegración de identidad y estructura
La estabilidad se busca alineando el sistema político con la identidad social.

**Forma de estabilidad:** Reactiva (reconstrucción desde la identidad)
El sistema se redefine en oposición al modelo anterior.

**Resultado en la identidad de Irán:** Integración de religión como base del Estado
Se consolida una nueva forma de cohesión estructural.

**Religión (rol en el sistema):** Islam chiita como eje central del poder político
Más del 95 por ciento de la población se identifica con el islam, ahora integrado en el Estado.

**Idioma (rol en el sistema):** Persa como continuidad cultural
Entre el 85 y el 95 por ciento utiliza el persa, manteniendo identidad nacional.

**Recurso principal (base de riqueza):** Petróleo y recursos estratégicos
El sistema mantiene su base económica, ahora bajo nueva estructura política.

**Capacidad económica (equivalente PIB):** Media — transición económica con ajustes estructurales
El sistema atraviesa un proceso de reorganización.

**Capacidad militar (proyección de poder):** Media — reorganización bajo nueva lógica
Las fuerzas se redefinen para responder al nuevo sistema.

**Dimensión territorial:** ~1,600,000 km² (~618,000 mi²)
Territorio consolidado bajo nueva estructura política.

**Población estimada:** ~35–40 millones
Con alta movilización social durante el proceso revolucionario.

## Análisis del sistema

En este periodo, el sistema no se ajusta.

Se redefine.

La Revolución Iraní no es solo un cambio político. Es una respuesta directa a la desconexión acumulada entre la estructura del Estado y la identidad de la sociedad. El modelo Pahlavi había intentado modernizar el sistema, pero lo hizo sin integrar completamente sus elementos internos.

El resultado fue desequilibrio.

Y el desequilibrio genera ruptura.

El movimiento revolucionario logra algo clave: alinear nuevamente el sistema con su identidad. La religión, que había sido desplazada, vuelve a ocupar el centro. Pero ahora no como elemento cultural.

Como estructura de poder.

Esto cambia completamente la lógica.

El sistema no busca parecerse a modelos externos.

Busca ser coherente consigo mismo.

La economía no desaparece, pero entra en transición. El sistema mantiene sus recursos, pero debe reorganizar su uso bajo una nueva estructura política.

La capacidad militar también se redefine. No se trata solo de defensa, sino de proteger el nuevo sistema. Esto introduce una lógica distinta de operación.

Aquí aparece el elemento clave.

La legitimidad.

Caso conceptual.

Un sistema puede tener estructura, recursos y capacidad militar, pero si no es percibido como legítimo por su propia población, pierde estabilidad. En cambio, un sistema alineado con su identidad puede sostenerse incluso bajo presión.

El primer modelo impone.

El segundo integra.

Eso es lo que ocurre aquí.

La estabilidad deja de depender de modernización o control.

Depende de legitimidad interna.

La lección es clara.

Un sistema que logra alinear su estructura con su identidad puede reconstruirse incluso después de una ruptura total.

Para entender a Irán hoy, este momento es fundamental. Define la base del sistema actual. La integración de religión, política e identidad no es un elemento secundario.

Es el núcleo.

Intentar analizar el sistema sin considerar esta integración genera errores.

Pero entenderla permite interpretar su comportamiento.

El sistema no es una continuidad del anterior.

Es una redefinición.

Y esa redefinición establece un nuevo equilibrio.

Pero también introduce nuevas tensiones.

El siguiente paso no es la estabilidad completa.

Es la operación de este nuevo sistema en un entorno global.

Ahí comienza la etapa actual.

Y con ella, la negociación cambia nuevamente.

Ya no se trata de definir el sistema.

Se trata de interactuar con él.
