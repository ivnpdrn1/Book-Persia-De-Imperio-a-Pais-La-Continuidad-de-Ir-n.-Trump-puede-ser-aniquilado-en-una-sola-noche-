# Capítulo 8

## La Cohesión del Sistema

**224–651 d.C.**

El sistema vuelve a concentrarse. Pero no vuelve igual.

Después de siglos de estabilidad basada en el equilibrio bajo los partos, aparece una nueva necesidad. El sistema es resistente, pero no es lo suficientemente eficiente para enfrentar entornos más exigentes. La falta de centralización limita la capacidad de respuesta y reduce la coordinación.

En este contexto surgen los sasánidas.

No intentan simplemente reemplazar el modelo anterior. Buscan corregirlo. Entienden que la flexibilidad permitió resistir, pero también impidió alcanzar un mayor nivel de organización.

El poder vuelve a concentrarse. El Estado se fortalece y la estructura se define con mayor claridad. Sin embargo, esta centralización no se basa únicamente en el control. Se apoya en un elemento clave: la cohesión.

La religión deja de ser solo un factor de diversidad tolerada y se convierte en eje del sistema. El zoroastrismo pasa a ser la referencia dominante, seguido por aproximadamente entre el 70 y el 90 por ciento de la población en sus distintas formas. Otras creencias continúan existiendo, pero con menor influencia.

Esto genera un cambio importante. La religión ya no solo conecta comunidades. Conecta al Estado.

El idioma también se alinea con esta lógica. El persa medio se consolida como lengua dominante en la administración y en la cultura. Se estima que entre el 50 y el 70 por ciento de la población lo utiliza directamente o como referencia cultural, mientras que las lenguas locales continúan presentes en menor medida.

La economía se fortalece. El sistema se reorganiza con una estructura más eficiente. Se establecen impuestos más claros, se mejora la producción agrícola y se mantiene un nivel activo de comercio. La capacidad económica es alta. Aunque no alcanza el nivel máximo del Imperio Aqueménida, el sistema logra una mayor coherencia interna.

La capacidad militar también cambia. Se vuelve organizada, centralizada y disciplinada. Se estima que podían movilizar entre 100,000 y 150,000 soldados. Más allá del número, la clave es la estructura. El ejército funciona como una extensión del Estado.

En el comercio, el territorio mantiene su posición estratégica. Continúa siendo un punto clave entre Oriente y Occidente, controlando rutas importantes que conectan diferentes regiones. Esto le permite mantener influencia sin depender completamente de factores externos.

El conocimiento se formaliza más. Aparecen estructuras organizadas que permiten al Estado utilizar el conocimiento como herramienta de administración y control. El nivel educativo en las élites es alto, lo que mejora la capacidad de gestión.

Para entender la escala, es importante observar sus dimensiones. El territorio se estima entre 3,000,000 y 3,500,000 de kilómetros cuadrados, equivalentes aproximadamente a entre 1,158,000 y 1,351,000 millas cuadradas. La población se calcula entre 20 y 25 millones de personas, con entre el 75 y el 85 por ciento viviendo en zonas rurales, aunque las ciudades continúan creciendo.

Aquí aparece el nuevo modelo.

Un sistema centralizado con cohesión interna.

Esta combinación le da una gran ventaja. Un sistema cohesionado responde mejor, se coordina con mayor eficiencia y resiste con mayor estabilidad.

Pero también introduce un nuevo riesgo.

La rigidez.

Cuando un sistema se vuelve demasiado estructurado, pierde capacidad de adaptación. Y cuando pierde flexibilidad, le resulta más difícil responder a cambios externos.

La estabilidad es alta.

La adaptabilidad disminuye.

La lección es clara.

La cohesión fortalece, pero también limita.

Para entender a Irán hoy, este momento es fundamental. Muestra el equilibrio entre dos fuerzas: control y flexibilidad. Demasiado control reduce la capacidad de adaptación, mientras que demasiada flexibilidad debilita la cohesión.

El sistema sasánida logra mantener ese equilibrio durante un tiempo.

Pero no de forma permanente.

Todo sistema altamente estructurado enfrenta un límite. Ese límite aparece cuando surge una forma distinta de organización, con una lógica diferente.

No necesariamente más fuerte.

No necesariamente más rápida.

Pero diferente.

Ese será el siguiente paso.

Cuando la estructura se enfrenta a una nueva lógica de organización.

Y cuando eso ocurre, la negociación cambia por completo.
